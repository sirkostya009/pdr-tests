# Тести ПДР

Безкоштовні тести з Правил дорожнього руху України. Без реєстрації, СМС чи реклами.

## 👉 [Відкрити тести — pdr-tests.vercel.app](https://pdr-tests.vercel.app)

Просто натисніть на посилання вище — нічого встановлювати не треба.

## Працює без інтернету

Цю веб апку можна **встановити як звичайний застосунок** і користуватися **повністю офлайн**.

- **Телефон:** у меню браузера оберіть «Додати на головний екран» / «Встановити застосунок».
- **Комп'ютер:** у полі адреси натисніть на значок встановлення.

## Що входить у «20 випадкових питань»

Випадковий тест збирає 20 питань із чотирьох тем:

- **10** питань — Загальні положення
- **4** питання — Будова і терміни (категорії B1, B)
- **4** питання — Надання першої медичної допомоги
- **2** питання — Основи безпечного водіння
